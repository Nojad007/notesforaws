This repo is for setting up an EC2 instance 


1. Creating EC2 Instance
   These are the steps to creating EC2 instance complete with a private subnet, a start up script, with the end goal   of using the DNS link to connect to this server


2.   Templates & Modified Template
   These are the steps to creating templates from the already created EC2 instances


3. VPC
   The photos in this folder shows the steps and/or things to think about and consider before making a VPC


4. Building a VPC
   The photos in this folder are showing the steps when creating a VPC with a Windows bastion host that will be connected to using port 3389 for RDP, security groups for both the bastion host and the private linux server


6. The following entries show the seamless process of setting up the:
   
   TG - Target Group
   IT - Instance Template
   ALB - Application Load Balancer
   ASG - Auto-Scale Group
